# ChessGameProject-GUI
Team : Just Me - Corey Hanna\
Fall 2025 - CS3354.R01
## Description
### Features
- Functioning GUI with click to move logic
- Text-based move log panel within GUI
- Ability to name and save game to .txt file
- Ability to load any saved game files
- New game button to clear board and log
- Undo functionality which restores previous board state and removes last move from log
- Functioning chess game with diagonal pawn movement, check, and checkmate logic
- Popup upon checkmate which declares winner and closes window

## Getting Started
### Dependencies
- JDK Version: 17 or higher
- Build Tool: IntelliJ IDEA
- Uses built-in Java libraries
- No external dependecies requied
### Installation and Execution
1. Clone respository
   - "git clone https://github.com/corey-h-txst/ChessGameProject-GUI/"
2. Open project in IntelliJ IDEA
   - Go to File -> Open and select downloaded project folder
3. Set up the SDK
   - Ensure that Java SDK (JDK 17 or higher) is configured
4. Build and Run
   - Open ChessGame.java class
   - Click green run button
